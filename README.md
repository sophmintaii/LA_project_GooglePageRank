## Prerequisites
```bash
pip install numpy
pip install pandas
pip install networkx
```
## Usage
two csv files are needed: one with adjacency lists for nodes, and one with nodes and their names
```
python3 main.py <file_adjacency_lists> <file_node_name>
```
